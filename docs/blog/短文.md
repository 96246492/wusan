# 论精彩的现代诗



“我默许了你的生长

默许了你的奇异，别致，你的怪诞不经

人们却聚起一百束目光

将你点燃，灼烧

我只能忍受炙烫的痛苦与你告别

从月亮脚下借走一千颗星星

将你埋葬

我的尚且幼嫩的翅膀

从此我再也无力飞翔 “
